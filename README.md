# TheTerminator-Themes
p10k TheTerminal Themes

## Prerequisites

[oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh)
[powerlevel10k](https://github.com/romkatv/powerlevel10k)

## Usage
> copy the THT theme into ~/.p10k.zsh and execute the comand
```
source ~/.p10k.zsh
```
> and enjoy
